🚀 Usage Instructions

1. Display Storefront

Create a WordPress page and use the custom "Store" template, or add the shortcode:

[amal_storefront per_page="12" show_filters="yes" show_search="yes"]

2. Display Item Details

Create a WordPress page with the "Item Detail" template, or use the shortcode:

[amal_item_detail item_id="123"]

Note: If no item_id is provided, it will look for ?item_id=123 in the URL

3. URL Structure

Store Page: /store/ Item Detail: /store/?item_id=123 With Filters: /store/?category=Food&search=dog&store_page=2

🎯 Acceptance Criteria Compliance

✅ Only Active Items Shown

The get_items() method includes WHERE is_active = 1 in all queries.

$where = "WHERE is_active = 1";

✅ Pagination & Filtering Work

Implemented with URL parameters and database LIMIT/OFFSET:

?store_page=2&category=Food&search=dog

✅ Item Detail from Database

Direct database queries load item details:

get_item($item_id) // Loads from wp_amal_items

🔧 Technical Implementation

Database Integration

The frontend directly queries the existing wp_amal_items table:

// Get active items with filtering SELECT * FROM wp_amal_items WHERE is_active = 1 AND category = %s AND (title LIKE %s OR description LIKE %s) ORDER BY created_at DESC LIMIT 12 OFFSET 0

WordPress Integration

Uses WordPress hooks and shortcodes for seamless integration:

// Register shortcodes add_shortcode('amal_storefront', array($this, 'render_storefront')); add_shortcode('amal_item_detail', array($this, 'render_item_detail')); // Handle AJAX requests add_action('wp_ajax_amal_add_to_cart', array($this, 'handle_add_to_cart'));
